# VoxLibre Lecteur Ebook IA

Application Flutter pour lire des fichiers EPUB et les synthétiser vocalement à l'aide de l'API Mistral.

## Instructions
Puisque le code Flutter a été généré sans lancer la compilation directement ("sans checker ou compiler"), pour terminer l'initialisation du projet localement, vous devez simplement utiliser la commande :

```bash
flutter create .
```

Cela génèrera les dossiers natifs manquants (`android/`, `ios/`, etc.). 

Le fichier `.env.example` illustre les variables d'environnement requises (`MISTRAL_API_KEY=...`).
